# SnapCut AI

SnapCut AI is an AI-powered background remover web application that allows users to remove image backgrounds in just one click. It provides a clean and simple interface where users can upload an image, process it using AI, and download the final transparent-background image.

## Features

- One-click AI background removal
- User-friendly and responsive UI
- Image upload and preview
- Download processed image
- User authentication
- Image history for logged-in users
- Backend API integration
- Fast and smooth image processing

## Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas

### Other Tools
- REST API
- JWT Authentication
- Cloudinary / Image storage
- GitHub
